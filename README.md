# wasm-othello

Othello based WebAssembly.  
Developed with [rust-webpack-template](https://github.com/rustwasm/rust-webpack-template).

![othello](./static/othello.png)

## Prerequisites

https://rustwasm.github.io/docs/wasm-pack/prerequisites/index.html

## Development setup

Dev server run

```bash
npm run start
```

Build

```bash
npm build
```
